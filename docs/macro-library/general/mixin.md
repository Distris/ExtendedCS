# Mixin

**Usable on**: `struct`, `class`

```cs title="Usage"
[Mixin(typeToMixin: typeof(TypeToMixIn))]
public partial class ToBeMixedInto {} 
```

Copies all members (fields, properties and methods) from `typeToMixin` into the type on which this
attribute is attached.

:::note
`typeToMixin` needs to be in the same assembly as the receiving type.
:::

## Example

```cs
public abstract partial class ToBeMixedIn {
  /// <summary>publicField</summary>
  public int publicField = 8;

  /// <summary>protectedField</summary>
  protected int protectedField = 5;

  /// <summary>privateField</summary>
  [SerializeField] int privateField = 4;

  /// <summary>publicProperty</summary>
  public int publicProperty => privateField;

  /// <summary>protectedProperty</summary>
  protected int protectedProperty => 3;

  /// <summary>privateProperty</summary>
  int privateProperty => 3;

  /// <summary>publicMethod</summary>
  public string publicMethod() => "Yeah!";
  
  /// <summary>protectedMethod</summary>
  protected string protectedMethod() => "Macros!";
  
  /// <summary>privateMethod</summary>
  string privateMethod() => "Cool!";
}
  
[Mixin(typeof(ToBeMixedIn))]
public partial class ToBeMixedInto {} 
```

Will generate:
```cs
public partial class ToBeMixedInto {
  // Generated by: MixinAttribute
  /// <summary>publicField</summary>
  public int publicField = 8;

  /// <summary>protectedField</summary>
  protected int protectedField = 5;

  /// <summary>privateField</summary>
  [SerializeField] int privateField = 4;

  [PublicAccessor] int _anotherPrivateField = 10;

  /// <summary>publicProperty</summary>
  public int publicProperty => privateField;

  /// <summary>protectedProperty</summary>
  protected int protectedProperty => 3;

  /// <summary>privateProperty</summary>
  int privateProperty => 3;

  /// <summary>publicMethod</summary>
  public string publicMethod() => "Yeah!";

  /// <summary>protectedMethod</summary>
  protected string protectedMethod() => "Macros!";

  /// <summary>privateMethod</summary>
  string privateMethod() => "Cool!";
  
  // Generated by: PublicAccessor
  /// <inheritdoc cref = "_anotherPrivateField"/>
  public int anotherPrivateField => _anotherPrivateField;
}
```
