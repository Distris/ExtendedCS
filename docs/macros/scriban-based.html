Scriban-Based Macros

You can write your own logic using the scriban template engine with [AttributeMacro].

Add this [AttributeMacro] attribute on your own custom attribute:

using System;
using GenerationAttributes;

[AttributeMacro(@"
  scriban code goes here
")]
public class MyAttribute : Attribute {}

Then your custom attribute ([MyAttribute] in this case) will generate new type members or extension methods when used on a type definition (class, struct or enum).

Scriban​

Scriban is a templating language, similar to PHP.

Reference links to the most useful documentation:

  • Language description — describes how blocks, control statements, comments, templating works.
  • Built-in functions — a comprehensive list of built-in functions available for you.

Examples​

Example of a field macro
using System;
using GenerationAttributes;

[
  AttributeMacro(@"
/// <summary>
/// <para>Sets the <see cref=""{{ field.name }}""/> value.</para>
/// </summary>
public {{ 
if field.is_static 
  'static' 
end 
}} void EDITOR_set{{ field.name | rename | pascal_case }}(
  {{ field.type | type_reduced_name }} value
) => {{ field.name }} = value;
  "), 
  AttributeUsage(AttributeTargets.Field)
]
public class EditorSetterAttribute : Attribute {}

class MyCharacterData {
  [SerializeField, EditorSetter] CharacterRole _type;
}
Example of a class macro
using System;
using GenerationAttributes;

[
  AttributeMacro(@"
public static TimeSpan {{ ourMethodBaseName }}(this short v) => 
  TimeSpan.{{ timeSpanMethodName }}(v);
public static TimeSpan {{ ourMethodBaseName }}(this ushort v) => 
  TimeSpan.{{ timeSpanMethodName }}(v);
public static TimeSpan {{ ourMethodBaseName }}(this int v) => 
  TimeSpan.{{ timeSpanMethodName }}(v);
public static TimeSpan {{ ourMethodBaseName }}(this uint v) => 
  TimeSpan.{{ timeSpanMethodName }}(v);
public static TimeSpan {{ ourMethodBaseName }}(this long v) => 
  TimeSpan.{{ timeSpanMethodName }}(v);
public static TimeSpan {{ ourMethodBaseName }}(this ulong v) => 
  TimeSpan.{{ timeSpanMethodName }}(v);
public static TimeSpan {{ ourMethodBaseName }}(this float v) => 
  TimeSpan.{{ timeSpanMethodName }}(v);
public static TimeSpan {{ ourMethodBaseName }}(this double v) => 
  TimeSpan.{{ timeSpanMethodName }}(v);

public static TimeSpan {{ ourMethodBaseName }}s(this short v) => 
  TimeSpan.{{ timeSpanMethodName }}(v);
public static TimeSpan {{ ourMethodBaseName }}s(this ushort v) => 
  TimeSpan.{{ timeSpanMethodName }}(v);
public static TimeSpan {{ ourMethodBaseName }}s(this int v) => 
  TimeSpan.{{ timeSpanMethodName }}(v);
public static TimeSpan {{ ourMethodBaseName }}s(this uint v) => 
  TimeSpan.{{ timeSpanMethodName }}(v);
public static TimeSpan {{ ourMethodBaseName }}s(this long v) => 
  TimeSpan.{{ timeSpanMethodName }}(v);
public static TimeSpan {{ ourMethodBaseName }}s(this ulong v) => 
  TimeSpan.{{ timeSpanMethodName }}(v);
public static TimeSpan {{ ourMethodBaseName }}s(this float v) => 
  TimeSpan.{{ timeSpanMethodName }}(v);
public static TimeSpan {{ ourMethodBaseName }}s(this double v) => 
  TimeSpan.{{ timeSpanMethodName }}(v);
  "),
  AttributeUsage(AttributeTargets.Class, AllowMultiple = true)
]
class GenTimeSpanHelpersAttribute : Attribute {
  /// <summary>Name of our extension, like 'milli'.</summary>
  public string ourMethodBaseName;
    
  /// <summary>Name on <see cref="TimeSpan"/>, like 'FromMilliseconds'.</summary>
  public string timeSpanMethodName;
}

[
  GenTimeSpanHelpers(
    ourMethodBaseName = "milli", 
    timeSpanMethodName = nameof(TimeSpan.FromMilliseconds)
  ),
  GenTimeSpanHelpers(
    ourMethodBaseName = "second", 
    timeSpanMethodName = nameof(TimeSpan.FromSeconds)
  ),
  GenTimeSpanHelpers(
    ourMethodBaseName = "minute", 
    timeSpanMethodName = nameof(TimeSpan.FromMinutes)
  ),
  GenTimeSpanHelpers(
    ourMethodBaseName = "hour", 
    timeSpanMethodName = nameof(TimeSpan.FromHours)
  ),
  GenTimeSpanHelpers(
    ourMethodBaseName = "day", 
    timeSpanMethodName = nameof(TimeSpan.FromDays)
  ),
] 
public static partial class TimeSpanExts {}

Parameters​

These parameters are always exposed to the scriban template:

type​

The current type of class/struct/enum or the containing type if your attribute is attached to a field or a property.

Parameter type: Type

fieldName​

Refers to a field value of field fieldName on your custom attribute.

Parameter type: mapped from the C# type using the following mapping.

C# typeScriban Type
boolboolean
stringstring
numeric types (int, uint, etc.)number
arrays (string[], int[], etc.)array
System.TypeType

For example:

[AttributeMacro(@"
{{
  # Note that by default the parameters are undefined.
  if !is_var_defined 'ParameterName'
    throw 'Please define a value for parameter ""ParameterName""'
  end
}}

public void newMethod(string {{ ParameterName }}) {
  System.Console.WriteLine($""Hi from {ParameterName}!"");
}
")]
public class MyAttribute : Attribute {
  public string ParameterName;
}

Field Parameters​

These parameters are only exposed if an attribute is added to a field:

[MyScribanMacro] public int MyField;

field​

The field that this macro is attached to. In the example case this would be the int MyField field.

Parameter type: Field

Property Parameters​

These parameters are only exposed if an attribute is added to a property:

[MyScribanMacro] public int MyProperty { get; set; }

property​

The property that this macro is attached to. In the example case this would be the int MyProperty property.

Parameter type: Property

Field or Property Parameters​

These parameters are exposed if an attribute is added to a field or property:

[MyScribanMacro] public int MyField;
[MyScribanMacro] public int MyProperty { get; set; }

They make writing templates which work on both fields and properties easier to write.

field_or_prop​

Parameter type: Field or Property.

The respective data from either a field or property parameters.

Types​

In addition to built-in scriban types, the compiler defines additional types.

EnumValue​

One of the defined cases in an enum MyEnum { Case1, Case2 } declaration.

It has these properties exposed to scriban:

NameTypeDescription
namestringThe fully qualified enum value name, for example global::MyNamespace.MyEnum.Case1.
short_namestringThe case name without the namespace and enum name, for example Case1.
valueobjectValue assigned to this enum member, usually an int. Also see information about underlying enum types.

Field​

The Field type has these properties exposed to scriban:

NameTypeDescription
namestringThe name of the field in C#, for example MyField for string MyField;
typeTypeThe type of the field in C#, for example string for string MyField;
is_read_onlybooleantrue if this is a readonly field, such as readonly string MyField;
is_constbooleantrue if this is a constant field, such as const string MY_CONSTANT = "value";
constant_valueobject?Constant value that we get from compiler API. Available for constant fields. For example, for const string MY_CONSTANT = "value"; this would be "value".
is_propertybooleanWhether this is a property. Always false.
is_fieldbooleanWhether this is a field. Always true.
is_methodbooleanWhether this is a method. Always false.
is_staticbooleanWhether this is a static field, like static int MyField;
is_privatebooleanWhether this is a private field, like private int MyField;
is_protectedbooleanWhether this is a protected field, like protected int MyField;
is_internalbooleanWhether this is an internal field, like internal int MyField;
is_publicbooleanWhether this is a public field, like public int MyField;
visibility_modifierstringOne of public, private, protected, internal or protected internal.
is_abstractbooleanAs fields cannot be abstract in C# this is always false.
is_virtualbooleanAs fields cannot be virtual in C# this is always false.

FieldOrProp​

An internal structure for the compiler, obtainable from create_field_or_prop function, useable in generate_record_members function.

Method​

The Method type has these properties exposed to scriban:

NameTypeDescription
namestringThe name of the the method in C#, for example ComputeName for string ComputeName();
return_typeTypeThe return type of the method in C#, for example string for string ComputeName();
is_read_onlybooleanIndicates whether the method is readonly, i.e. whether the 'this' receiver parameter is 'ref readonly'.

Returns true for readonly instance methods and accessors and for reduced extension methods with a 'this in' parameter.
parametersParameter[]Array of C# parameters provided to this method.
type_parametersTypeParameter[]Array of C# type parameters provided to this generic method. If the method is non-generic, this is empty.
is_propertybooleanWhether this is a property. Always false.
is_fieldbooleanWhether this is a field. Always false.
is_methodbooleanWhether this is a method. Always true.
is_staticbooleanWhether this is a static method, like static int CalculateAge();
is_privatebooleanWhether this is a static method, like private int CalculateAge();
is_protectedbooleanWhether this is a protected method, like protected int CalculateAge();
is_internalbooleanWhether this is an internal method, like internal int CalculateAge();
is_publicbooleanWhether this is a public method, like public int CalculateAge();
visibility_modifierstringOne of public, private, protected, internal or protected internal.
is_abstractbooleanTrue if this is an abstract method, such as abstract int CalculateAge();
is_virtualbooleanTrue if this is an virtual method, such as public virtual int CalculateAge();

Parameter​

Parameter provided to an indexer property (see parameters on Property) or a regular Method.

Example of an indexer property
class SampleCollection<T> {
  // Declare an array to store the data elements.
  private T[] arr = new T[100];

  // Define the indexer to allow client code to use [] notation.
  public T this[int idx] {
    get { return arr[idx]; }
    set { arr[idx] = value; }
  }
}

It has these properties exposed to scriban:

NameTypeDescription
namestringThe name of the parameter, for example idx.
typeTypeThe type of the parameter, for example int.

Property​

The Property type has these properties exposed to scriban:

NameTypeDescription
namestringThe name of the property in C#, for example MyProp for string MyProp { get; set; }
typeTypeThe type of the property in C#, for example string for string MyProp { get; set; }
is_read_onlybooleanWhether the property can only be read, for example string MyProp { get; }
is_write_onlybooleanWhether the property can only be written, for example string MyProp { set; }
has_getterbooleanWhether the property has the get; part, for example string MyProp { get; } or string MyProp { get; set; }
has_setterbooleanWhether the property has the set; part, for example string MyProp { set; } or string MyProp { get; set; }
parametersParameter[]If this property is an indexer (such as int this[string key1, string key2] => ...;), this returns an array of the parameters provided to the indexer.
is_propertybooleanWhether this is a property. Always true.
is_fieldbooleanWhether this is a field. Always false.
is_methodbooleanWhether this is a method. Always false.
is_staticbooleanWhether this is a static property, like static int MyProp { get; set; }
is_privatebooleanWhether this is a static property, like private int MyProp { get; set; }
is_protectedbooleanWhether this is a protected property, like protected int MyProp { get; set; }
is_internalbooleanWhether this is an internal property, like internal int MyProp { get; set; }
is_publicbooleanWhether this is a public property, like public int MyProp { get; set; }
visibility_modifierstringOne of public, private, protected, internal or protected internal.
is_abstractbooleanTrue if this is an abstract property, such as abstract int MyProp { get; set; }
is_virtualbooleanTrue if this is an virtual property, such as public virtual int MyProp { get; set; }

Type​

The Type type has these properties exposed to scriban:

NameTypeDescription
namestringThe fully qualified type name, for example global::System.Collections.Generic.List<int>.
short_namestringThe type name without the namespace and generic parameters, for example List.
visibility_modifierstringOne of public, private, protected, internal or protected internal.
underlying_enum_typeType?If this Type is an enum, then this will have the underlying type of that enum.

For example, enum MyEnum : byte { Case1, Case2 } has an underlying type of byte.

enums that do not have an underlying type specified explicitly will use int as its underlying type.

TypeParameter​

A type parameter provided to a generic Method.

Example of a generic method
TDispatcher DoDispatch<TDispatcher, TEvent>(
  TDispatcher dispatcher, Action<TEvent> onEvent
)
  where TDispatcher : IDispatcher
{
  // ...
}

It has these properties exposed to scriban:

NameTypeDescription
namestringThe name of the generic parameter, for example TDispatcher or TEvent.
additional_constraintstring?Indicates whether the generic parameter has any of these constraints:
  • struct — for example in where TEvent : struct
  • class — for example in where TEvent : class
  • unmanaged — for example in where TEvent : unmanaged
  • null otherwise
has_constructor_constraintbooleantrue if the generic parameter has the where TEvent : new() constaint.
constraint_typesType[]Array of types that constrain this generic parameter, for example where TEvent : IReadable, IWritable.

Functions​

In addition to the built-in functions we additionally add these functions to scriban templates.

Case Changing Functions​

These functions from Soltys.ChangeCase library are exposed to the scriban template.

  • string sentence_case(string input, string replacement = " ")
  • string camel_case(string input)
  • string pascal_case(string input)
  • string upper_case_first(string input)
  • string param_case(string input)
  • string dot_case(string input)
  • string swap_case(string input)
  • string title_case(string input)
  • string snake_case(string input)
  • string constant_case(string input)

add_extensions​

Function Signature: void add_extensions(string extensionsCode)

Pass a string that contains extension methods, they will be parsed and added to a separate static class.

You can use capture feature of scriban engine to do that conveniently.

Example
{{ capture extensions }}
  {{ exts_type_name = type.name }}

  /// <summary>Converts a value to a single flag.</summary>
  public static {{exts_type_name}}Flags toFlags(this {{exts_type_name}} v) =>
    ({{exts_type_name}}Flags) (1{{shiftSuffix}} << (int) v);
{{ end }}

{{ add_extensions extensions }}

add_interface​

Function Signature: void add_interface(string interfaceName)

Adds an interface to a generated partial class.

Example
{{
  add_interface 'IEquatable<' + (typeFor | type_reduced_name) + '>'
}}

add_using​

Function Signature: void add_using(string usingName)

Adds using X; C# directive to the generated file.

Example
{{
  add_using 'FPCSharpUnity.core.json'
}}

create_field_or_prop​

Function Signature: FieldOrProp create_field_or_prop(Type type, string name)

Creates a FieldOrProp data structure from a field or property with the name name on a type. This data structure can only be passed to generate_record_members(#generate_record_members) function.

fdqn_last​

Function Signature: string fdqn_last(string identifier)

Returns the string without the namespace and generic parameters.

Example
{{
  name = fdqn_last 'System.Collections.Generic.List<int>'
  # name = 'List'
}}

find_type​

Function Signature: Type? find_type(string fdqn, Type? type_for_assembly = null)

Tries to find a type in either a single assembly or whole compilation.

If type_for_assembly is provided, the search is only conducted in that assembly, otherwise all available assemblies are searched.

You can use the type_get_full_metadata_name function to get the fdqn that is suitable for this function.

Returns null if not found.

Example
{{
  maybeAssetRefFromType = 'Quantum.' + assetRefFromName | find_type
}}

generate_record_members​

Function Signature: 

void generate_record_members(
  (Field|Property|FieldOrProp)[] fields_and_props,
  boolean generate_comparer = true,
  boolean generate_to_string = true,
  boolean generate_get_hash_code = true,
  boolean constructor_flags = GenerationAttributes.ConstructorFlags.Default,
)

Generates record members (constructors and other functions) on the current type for fields_and_props that were given.

is_var_defined​

Function Signature: boolean is_var_defined(string variable)

Returns whether a variable is defined in scriban template root scope.

This is very useful in dealing with default parameters in the attributes.

Example
[AttributeMacro(@"
{{
  if !is_var_defined 'isSomeName'
    isSomeName = '__unsafeIsSome'
  end

  # ...
}}
")]
public class NullableTAttribute : Attribute {
  /// <summary>Name for the 'bool isSome' field. `__unsafeIsSome` by default.</summary>
  public string isSomeName;

  // ...
}

rename_as_public_accessor​

Function Signature: string rename_as_public_accessor(string identifier)

Renames the string according to [PublicAccessor] naming rules.

Example
{{
  public_name = rename_as_public_accessor '_MyField'
  # public_name = 'MyField'
}}

throw​

Function Signature: void throw(string message)

Fails the code generation and passes the error message to the compiler output.

Example
{{
  if YourParameter > 6
    throw 'YourParameter can not be more than 6, you provided ' + YourParameter + '!'
  end
}}

type_arguments​

Function Signature: Type[] type_arguments(Type type)

Returns an array of generic arguments if they exist on this type.

Example
{{
  # type = typeof(System.Collection.Generic.List<int>)

  type_args = type_arguments type
  # type_args = [typeof(int)]
}}

type_enum_value​

Function Signature: object type_enum_value(Type type, string name)

Gets the underlying constant value for the enum case with the name.

Example
{{
  # Given the definition:
  #   enum MyEnum { Case1 = 5, Case2 = 10 }
  #
  # type = typeof(MyEnum)

  enum_value = type_enum_value type 'Case1'
  # enum_value = 5
}}

type_enum_values​

Function Signature: EnumValue[] type_enum_values(Type type)

Returns an array of enum values. Only available if type is an enum type.

Example
{{
  # Given the definition:
  #   enum MyEnum { Case1, Case2 }
  #
  # type = typeof(MyEnum)

  enum_values = type_enum_values type
  # enum_values = [EnumValue(MyEnum.Case1), EnumValue(MyEnum.Case2)]
}}

type_get_all_fields​

Function Signature: 

Field[] type_get_all_fields(
  Type type,
  boolean public = true,
  boolean private = true,
  boolean protected = true,
  boolean instance = true,
  boolean static = false,
  boolean const = false
)

Gets all fields of a Type. You can filter the list based on parameters.

Use the named arguments syntax to set up the filters.

Example
{{
  fields = type_get_all_fields my_type static:true
}}

type_get_all_methods​

Function Signature: 

Method[] type_get_all_methods(
  Type type,
  boolean public = true,
  boolean private = true,
  boolean protected = true,
  boolean instance = true,
  boolean static = false,
  // Whether to include methods from base types.
  boolean include_base_types = false,
  // Whether to include methods from the implemented interfaces.
  boolean include_interfaces = false
)

Gets all methods of a Type. You can filter the list based on parameters.

Example
{{
  methods = type_get_all_methods my_type static:true
}}

type_get_all_properties​

Function Signature: 

Property[] type_get_all_properties(
  Type type,
  boolean public = true,
  boolean private = true,
  boolean protected = true,
  boolean instance = true,
  boolean static = false,
  // Whether to include methods from base types.
  boolean include_base_types = false,
  // Whether to include methods from the implemented interfaces.
  boolean include_interfaces = false
)

Gets all properties of a Type. You can filter the list based on parameters.

Example
{{
  properties = type_get_all_properties my_type static:true
}}

type_get_descendants_in_assembly​

Function Signature: Type[] type_get_descendants_in_assembly(Type type, Type type_for_assembly = null, boolean collect_indirect_descendants = false)

Gets all descendant types (types that extend the specified type) that are contained in a single assembly.

The assembly to check is taken from the type_for_assembly parameter. If type_for_assembly is not provided, then we take the assembly in which type is defined.

  • collect_indirect_descendants - if false, this function will only collect direct (Level 1) descendants.
note

This function is not optimized so do not call it often.

type_get_field​

Function Signature: Field type_get_field(Type type, string fieldName)

Gets a field of a Type with a specified name.

Example
{{
  field = type_get_field my_type 'MyField'
}}

type_get_full_metadata_name​

Function Signature: string type_get_full_metadata_name(Type type)

Gets the full C# metadata name that can be used in the find_type function.

Implementation is taken from this StackOverflow answer.

Example
{{
  type_metadata_name = type_get_full_metadata_name my_type
}}

type_get_property​

Function Signature: Property type_get_property(Type type, string propertyName)

Gets a property of a Type with a specified name.

Example
{{
  field = type_get_property my_type 'MyProperty'
}}

type_get_record_members​

Function Signature: 

{
  "all": (Field|Property)[],
  "fields": Field[],
  "properties": Property[],
} type_get_record_members(Type type)

Returns an object with 3 properties (all, fields, properties) containing members of a type retrieved using the same logic as a [Record] would.

Example
{{
  for member in (type_get_record_members of_type).all
    ...
  end
}}

type_reduced_name​

Function Signature: string type_reduced_name(Type type)

Returns a shortened string representation of a Type name that is usable in the generated code without clashing with other types.

This is very useful to make generated code more readable by replacing types like global::MyNamespace.MyClass.MyInnerClass with just MyClass.MyInnerclass.

However, the substitution may not always be possible in the current context, then this just returns Type.name.
